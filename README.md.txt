Automation Testing Maven Project
Welcome to the Automation Testing Maven Project! This repository contains a comprehensive test automation framework designed for web applications, leveraging Selenium, TestNG, and Maven. Below you'll find an overview of the project, setup instructions, and guidelines for running tests and contributing to the project.



Project Overview
This project aims to provide a scalable and robust automation testing framework. It utilizes Selenium WebDriver for browser automation, TestNG for test orchestration, and Maven for build automation and dependency management.

Features
Selenium WebDriver for browser automation
TestNG for test execution and reporting
Maven for dependency management and build automation
Page Object Model (POM) for maintaining clean and manageable code
Data-driven testing
